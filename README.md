Портал идей
Веб-приложение для сбора и управления идеями и предложениями сотрудников.

Основные возможности
Для пользователей
📝 Добавление новых идей с описанием проблемы и решения

📎 Прикрепление файлов (изображения, документы)

🏷️ Выбор категории для идеи

🔍 Просмотр опубликованных идей с фильтрацией и поиском

📧 Уведомления о статусе идеи (если указан email)

Для модераторов
👥 Два предустановленных модератора с разными правами

✅ Модерация идей: одобрение, отклонение, изменение статуса

📊 Панель управления с фильтрами и статистикой

📥 Экспорт идей в Excel

🗂️ Управление категориями идей
